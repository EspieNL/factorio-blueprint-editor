## Next Steps

To activate Space Age support, run:

```bash
cd packages/exporter
cargo run --release
```

This will:
1. Download Factorio 2.0.68 (with Space Age)
2. Extract all game data including Space Age groups
3. Generate updated `data/output/data.json`
4. Compress and optimize all icons
